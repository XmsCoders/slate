    Contacts

    Récuperer les contacts

    Requête d'example :

    curl -v -X GET http://api.multimediabs.com/v1/groups/4633a4fe-5544-4ac9-95ac-9e1ffc4751bc/contacts \
    -H "Content-Type:application/json" \
    -H "Authorization: Bearer [Access-Token]"

    Cette requête retourne le Json suivant :

    {
    "content":[
        {
            "id":"f12edd46-96b8-41d9-baa7-58f42daa9301",
            "name":"Boyer",
            "first-name":"Mael"
        },
        {
            "id":"54032d6f-19cd-4049-8cfe-bd80a563c911",
            "name":"Le gall",
            "first-name":"Ethan"
        },
        {...},
        {
            "id":"372c1ede-03d5-41d4-bd2c-0471a3c371be",
            "name":"Morel",
            "first-name":"Alexis"
        }
    ],
    "last":false,
    "first":true,
    "totalPages":3,
    "totalElements":25,
    "numberOfElements":25,
    "size":10,
    "number":0
}

    Ceendpoint permet de récuperer tous les contacts d'un groupe donné.

    HTTP Request

    GET http://api.multimediabs.com/v1/api/groups/{id-group}/contacts

    Query Parameters

    Parameter Type Description
    id-group String L'identifiant unique du groupe.

    Headers Parameters

    Header Value Description
    Content-Type application/json content-type utilisé.
    Authorization Bearer [Access-Token] Permet de vous authentifier sur l'api.
    N'oubliez pas de remplacer [Access-Token] par votre token.
